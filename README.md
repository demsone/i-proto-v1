# Terpene Intelligence v1.0 — Prototype

Terpene-driven flower recommendation engine. Answer 5 questions, get matched to the right strain based on terpene pharmacology.

## Deploy

1. Push to GitHub
2. Connect to [Vercel](https://vercel.com)
3. Auto-deploys on every push

## Local Dev

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
